# 舆论传播机制与意见领袖分析研究报告

## 基于高市早苗相关推文的社交媒体舆论研究

---

## 一、研究背景与目的

### 1.1 研究背景

社交媒体的快速发展使得公众舆论的形成和传播模式发生了根本性变革。Twitter（现X平台）作为全球最具影响力的社交媒体之一，已成为政治人物与公众互动、舆论形成和传播的核心场域。日本政治家高市早苗作为令和年间活跃的重要政治人物，其相关推文数据涵盖了从自民党总裁选举到担任首相期间的多个关键时间节点，为研究舆论传播机制提供了丰富且具有时效性的素材。

### 1.2 研究目的

本研究旨在：

1. **揭示舆论传播机制**：分析信息在社交网络中的传播路径和模式，识别关键传播节点与结构特征
2. **识别意见领袖**：通过 PageRank、介数中心性等多维度网络指标，识别舆论场中的关键影响者
3. **量化情感倾向**：基于关键词匹配方法，评估公众对高市早苗的情感态度分布及其变化规律
4. **追踪话题演化**：发现和追踪核心讨论话题及其随时间的演变趋势
5. **构建分析系统**：搭建具有可视化界面的舆论分析平台，实现从数据处理到结果展示的全流程覆盖

## 二、数据收集与处理

### 2.1 数据来源

本研究使用 Twitter/X 平台公开推文数据，以高市早苗（Takaichi Sanae）为关键词进行检索和采集。数据集基本信息如下：

| 数据集标识 | 语言构成 | 时间范围 | 原始数据量 |
|-----------|---------|---------|-----------|
| takaichi_chinese | 中文为主（日语93%，中文4.6%，其他2.4%） | 2025-10 ~ 2026-05 | 271,852 条 |

### 2.2 数据字段

原始数据包含丰富的用户信息和推文特征：

- **用户属性**：用户ID、昵称、粉丝数、关注数、发推数、账户创建时间
- **互动指标**：点赞数（favorite_count）、转发数（retweet_count）、回复数（reply_count）、引用数（quote_count）、浏览量（view_count）
- **传播信息**：回复关系（in_reply_to）、引用关系（quoted_status）、对话链ID（conversation_id_str）
- **内容实体**：话题标签（hashtags）、@提及（mentions）、URL、媒体附件（media）
- **时间信息**：推文发布时间（created_at）

### 2.3 数据预处理

数据预处理流程包含以下四个步骤：

1. **数据清洗**：过滤无效记录（含删除标记的推文、空白文本）、规范化 URL 和特殊字符
2. **字段标准化**：统一日期时间格式（ISO 8601），提取用户名、互动数等关键字段
3. **数据转换**：JSONL 格式转换为结构化 CSV，构建设施网络分析所需的邻接表
4. **特征工程**：计算用户参与度得分（engagement_score = favorites + retweets × 2），提取时间维度特征（小时/日/周分布）

## 三、舆论分析维度与方法

### 3.1 传播网络分析

**网络构建方法**：

- 使用 NetworkX 构建有向加权图，节点为用户，边的方向代表信息流动方向
- 边类型包括：回复关系（reply）、引用关系（quote），权重为互动次数
- 对重复互动进行去重，保留唯一传播关系

**网络分析方法**：

- **度中心性（Degree Centrality）**：衡量节点的直接连接数量，入度反映"被动传播"能力，出度反映"主动传播"能力
- **PageRank 算法**：基于随机游走的节点重要性评估，考虑了网络拓扑结构中的间接影响
- **介数中心性（Betweenness Centrality）**：识别信息传播路径中的"桥梁"节点，该类节点控制着信息在社区间的流通

### 3.2 意见领袖识别

**评估指标体系**：

| 指标 | 权重 | 说明 |
|------|------|------|
| PageRank | 40% | 综合网络影响力评估，考虑了间接连接 |
| 介数中心性 | 30% | 信息传播路径中的中介重要性 |
| 度中心性 | 30% | 直接互动与传播能力 |

**综合影响力公式**：

$$InfluenceScore = 0.4 \times PageRank + 0.3 \times Betweenness + 0.3 \times DegreeCentrality$$

该公式通过加权平均将三个标准化后的网络指标融合为单一的影响力得分，用于意见领袖的最终排名。

### 3.3 情感分析

**方法**：基于关键词匹配的情感分类方法，针对日语和中文分别构建情感词典。

- **正面情感**：包含「支持」「加油」「伟大」「素晴らしい」等正向词汇
- **负面情感**：包含「反对」「失望」「糟糕」「最悪」「やめろ」等负向词汇
- **中立情感**：无明显情感倾向的客观陈述或信息转发

### 3.4 话题聚类

**方法**：

- 话题标签（Hashtag）频率统计，识别高频讨论主题
- 基于 jieba 分词库的关键词提取，捕捉文本内容主题
- 月度话题变化追踪，识别话题的生命周期特征

### 3.5 时间序列分析

**分析粒度**：日、周、月三级时间粒度
**核心指标**：推文数量、互动量（favorites + retweets）、活跃用户数
**分布模式**：周内分布（周一至周日）、日内分布（0-24小时）

## 六、研究发现与讨论

### 6.1 主要发现

**发现一：舆论场呈"核心-中介-边缘"三层结构**

- @takaichi_sanae 本人作为绝对中心节点，PageRank 值是第二名的近2倍，占据信息传播的核心位置
- 媒体账号（@47news、Sankei_news、nikkei 等）构成中介层，凭借高粉丝基数在信息分发中发挥关键作用
- 高互动草根用户（如 @ashitawawatashi）通过持续参与，在局部网络中建立影响力，形成"边缘突围"现象

**发现二：情感态度以中立占主导，支持与反对力量悬殊**

- 78% 的中立态度表明多数用户处于"信息搜集阶段"，尚未形成明确立场
- 正面情感（16.9%）是负面情感（5.1%）的3.3倍，这一正负比在政治人物相关话题中较为罕见，可能与数据集采集时间段（2025-2026年，高市担任首相期间）有关
- 批评性标签虽在数量上少于支持性标签，但情感强度更高（使用"最悪""国難""やめろ"等极端表达）

**发现三：话题高度集中于人物本人，宗教关联和政治立场是主要争议维度**

- 话题分布呈现"人物中心化"特征：几乎所有高频标签均围绕高市早苗本人展开
- 争议主要沿两条轴线展开：①宗教关联轴（统一教会）②政治表现轴（政策/执政能力）
- 话题的多元性（支持与反对并存）表明该政治人物在公众认知中存在显著分歧

**发现四：舆论热度呈典型事件驱动模式**

- 2025年10月初（疑似总裁选举期间）出现数据期内最大峰值，单日互动量超千万
- 2026年2月初（总裁选举结果公布期）形成持续一周的高热度密集区
- 两次高峰的时间间隔（约4个月）反映了日本政治议程的阶段性特征

### 6.2 研究局限

1. **数据来源单一性**：仅使用 Twitter/X 平台数据，存在平台用户结构的固有偏差（如年轻用户偏多），无法代表全体公众意见
2. **语言处理局限**：日语情感分析依赖关键词匹配方法，对讽刺、反语、隐喻等复杂表达识别能力有限
3. **相关性而非因果性**：网络分析揭示的是结构相关性，无法直接推断传播因果关系
4. **时间窗口约束**：数据截止至2026年5月，研究结论具有时效性特征
5. **嵌入数据架构限制**：当前纯前端架构无法处理实时新增数据，分析结果为静态快照

### 6.3 未来展望

1. **多平台数据融合**：整合微博、微信公众号、Reddit、YouTube 评论等多源异构数据，构建更全面的舆论画像
2. **深度学习情感分析**：引入 mBERT 等多语言预训练模型，提升情感分类精度，特别是对讽刺和隐喻表达的识别能力
3. **实时流式处理**：采用 Kafka + Flink 架构，构建流式数据处理管线，实现分钟级舆论监测
4. **传播预测模型**：基于历史传播数据训练图神经网络（GNN）预测模型，提前识别潜在的病毒式传播路径
5. **可解释性增强**：将 SHAP 等模型解释方法引入意见领袖识别，提升分析结果的透明度和可信度

## 七、结论

本研究以高市早苗相关推文为研究对象，系统运用社交网络分析与自然语言处理方法，对社交媒体舆论的传播机制进行了多维度实证研究。主要结论如下：

1. **传播网络呈现以政治人物本人为核心的星型拓扑结构**，少数高 PageRank 节点控制着信息的主要流向，网络整体具备良好的传播可达性

2. **情感态度以中立为主导（78%），但批评性话题活跃**，支持声音（16.9%）在量上多于反对（5.1%），舆论整体处于态度形成的信息搜集阶段

3. **话题高度集中于政治人物本人，宗教关联和执政表现构成两大争议主线**，体现了社交媒体政治讨论的典型议题特征

4. **舆论热度呈事件驱动模式**，重要政治节点（总裁选举等）引发显著的传播峰值，反映了社交媒体舆论与现实政治事件的高度联动性

本研究通过构建可视化舆论分析平台，验证了将网络科学与自然语言处理方法相结合用于舆论研究的可行性，为数字时代的社会舆论监测提供了技术参考。
