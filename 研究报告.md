# 舆论传播机制与意见领袖分析研究报告

## 基于高市早苗相关推文的社交媒体舆论研究

---

## 一、研究背景与目的

### 1.1 研究背景

社交媒体的快速发展使得公众舆论的形成和传播模式发生了根本性变化。Twitter（现X平台）作为全球最具影响力的社交媒体之一，已成为政治人物与公众互动、舆论形成和传播的重要平台。日本政治家高市早苗作为令和年间的重要政治人物，其相关推文数据涵盖了从自民党总裁选举到担任首相期间的多个重要时间节点，为研究舆论传播机制提供了丰富的素材。

### 1.2 研究目的

本研究旨在：
1. **揭示舆论传播机制**：分析信息在社交网络中的传播路径和模式
2. **识别意见领袖**：通过多维度指标识别舆论场中的关键影响者
3. **量化情感倾向**：评估公众对高市早苗的情感态度分布
4. **追踪话题演化**：发现和追踪核心讨论话题及其变化趋势
5. **构建分析系统**：搭建可视化的舆论分析平台

## 三、舆论分析维度与方法

### 3.1 传播网络分析

**构建方法**：
- 使用NetworkX构建有向图
- 边类型包括：回复关系、引用关系
- 节点为用户，边权重为互动次数

**分析方法**：
- **度中心性**：衡量直接连接数量
- **PageRank**：基于随机游走的节点重要性评估
- **介数中心性**：识别信息传播的"桥梁"节点

### 3.2 意见领袖识别

**评估指标体系**：

| 指标 | 权重 | 说明 |
|------|------|------|
| PageRank | 40% | 网络影响力的综合评估 |
| 介数中心性 | 30% | 信息传播路径中的重要性 |
| 度中心性 | 30% | 直接互动能力 |

**公式**：

$$InfluenceScore = 0.4 \times PageRank + 0.3 \times Betweenness + 0.3 \times DegreeCentrality$$

### 3.3 情感分析

**方法**：基于关键词匹配的情感分类
- **正面**：包含支持、加油、伟大等正向词汇
- **负面**：包含反对、失望、糟糕等负向词汇
- **中立**：无明显情感倾向

### 3.4 话题聚类

**方法**：
- 话题标签频率统计
- 基于jieba的关键词提取
- 月度话题变化追踪

### 3.5 时间序列分析

**粒度**：日、周、月
**指标**：推文数量、互动量、活跃用户数
**模式**：周内分布、小时分布

## 六、研究发现与讨论

### 6.1 主要发现

1. **舆论场的"中心-边缘"结构**
   - @takaichi_sanae本人作为中心节点，占据绝对的信息传播优势
   - 媒体账号构成第二层影响力圈层
   - 草根用户通过高互动量可获得较大影响力

2. **情感态度的分化特征**
   - 中立态度占主导（78%），表明多数用户保持理性观望
   - 支持与反对均存在，但正面情感(16.9%)超过负面(5.1%)
   - 批评性话题标签使用频率较高，反映舆论场的争议性

3. **话题的高度集中与多元化并存**
   - 核心话题高度集中于"高市早苗"本人
   - 同时存在支持、反对、宗教争议等多个子话题
   - 话题标签反映日本政治舆论的复杂性

4. **时间分布的事件驱动特征**
   - 舆论热度与重要政治事件高度相关
   - 2025年10月初和2026年2月初为两个显著高峰
   - 周内分布相对均匀，周二和周六略高

### 6.2 研究局限

1. **数据来源单一**：仅使用Twitter数据，可能存在采样偏差
2. **语言处理局限**：主要使用中文数据，日语情感分析精度有限
3. **因果推断缺失**：相关性分析无法推断因果关系
4. **时间窗口限制**：数据截止到2026年5月，可能已过时

### 6.3 未来展望

1. **多平台数据融合**：整合微博、微信公众号等多平台数据
2. **深度学习模型**：引入BERT等预训练模型提升情感分析精度
3. **实时监测系统**：构建流式数据处理架构，实现实时舆论监测
4. **传播预测**：基于历史数据预测舆论传播趋势

## 七、结论

本研究以高市早苗相关推文为研究对象，系统分析了社交媒体舆论的传播机制。研究表明：

1. **传播网络呈现中心化特征**：核心节点（意见领袖）对信息传播具有决定性影响
2. **情感态度总体偏正面**：但批评性话题活跃，反映舆论的多元性
3. **话题高度集中于政治人物本人**：宗教关联和政党立场是主要争议焦点
4. **舆论热度事件驱动**：重要政治节点引发舆论高峰

通过构建的舆论分析系统，实现了从数据处理到可视化展示的完整流程，为舆论研究和监测提供了技术支撑。
